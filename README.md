# Gaurang Events

Responsive React website for an event management business offering:

- Corporate gatherings
- Haldi
- Mehndi
- Get together
- Shadi
- Sagai

It includes:

- Hero landing page
- Event showcase cards
- Services and gallery sections
- `Get Quote` and `Enquire Now` enquiry form
- Unsplash images
- Mobile responsive layout

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
